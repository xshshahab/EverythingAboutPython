[ Example of list in Python ]

- A list is an ordered collection of items that is mutable, meaning you can change its contents. Lists are defined using square brackets [] and can hold items of any data type.

[ Example of tuple in Python ]

- A tuple is an ordered, immutable collection of items, meaning its contents cannot be changed after it is created. Tuples are defined using parentheses ().

[ Example of dictionary in Python ]

- A dictionary (or dict) is a mapping data type in Python that stores data as key-value pairs. It is unordered, mutable, and does not allow duplicate keys.
